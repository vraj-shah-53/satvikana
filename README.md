# 🌱 Satvikana - Makhana E-Commerce Store

## Overview

Satvikana is a full-stack e-commerce platform developed for selling premium Makhana products online. The platform provides customers with a seamless shopping experience while offering administrators tools to manage products and orders efficiently.

---

## Features

### Customer Features

* Browse premium Makhana products
* Product search and filtering
* Product detail pages
* Shopping cart functionality
* User authentication
* Order placement
* Order history tracking
* Healthy recipes section
* Responsive design

### Admin Features

* Product management
* Order management
* Order status updates
* Category management

## Installation

### Clone Repository

```bash
git clone https://github.com/vraj-shah-53/satvikana.git
cd satvikana
```

### Backend Setup

```bash
python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

cd satvikana_backend

python manage.py migrate
python manage.py runserver
```

### Frontend Setup

```bash
cd satvikana_frontend

npm install

npm run dev
```
